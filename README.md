# Chore & Screen Time Rules

A structured, child-friendly rules document designed for a specific 10-year-old boy, built as a styled HTML page with external CSS stylesheets.

## What This Is

This project is a household rules document covering daily chores, screen time, behavioral expectations, and incentives. It was written to be clear and easy to understand for a child with autism, using plain language, short rules, visual color-coding, and a consistent structure throughout.

The document covers:

- Daily chore requirements and screen time rewards
- A modified rule set for Saturdays, accounting for the family's weekly day out
- A weekend bonus screen time system tied to weekly chore consistency and behavior
- A weekly monetary reward to further encourage completion of all chores
- Mom's personal quiet hours and what constitutes a real emergency
- A dispute resolution process called "The Council"
